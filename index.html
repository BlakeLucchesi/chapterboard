ChapterBoard

Fraternity and Sorority communication and organization tools. The official repository of the open-sourced code from www.chapterboard.com

Credits

ChapterBoard was created and operated by Blake Lucchesi and Alexandra Harris. The logo and iPhone application icons were designed by our good friend Lance Alton Troxel of This By Them.

The ChapterBoard web application was built upon the Kohana PHP MVC Framework (version 2.3.x). It was hosted on a server running Ubuntu Linux with the nginx web server and MySQL database server.

Requirements

  • Apache or Nginx
  • PHP 5.2+
  • MySQL 5.x

For additional setup and installation please read the project wiki on GitHub.

Support or Contact

Having trouble? Please check out the issue queue on GitHub, but do keep in mind that the source code is released as is, without any guarantee or support. If the project maintainers have time, they'll do their best to help you out.
