Requirements

  • Apache or Nginx
  • PHP 5.2+
  • MySQL 5.x

Deployment Notes:

When ChapterBoard was in production we ran it on Nginx and PHP 5.2. PHP 5.3 support has tested on development servers so it should work if you have a more recent version of PHP.

Memcached is a great way to speed up cache access but its not essential as file based caching is enabled by default.

Support or Contact

Having trouble? Please check out the issue queue on GitHub, but do keep in mind that the application is released as is, without any guarantee or support. If the project maintainers have time, they'll do their best to help you out.
